# Algorithm Katas

Problem-solving practice for algorithm interviews and implementation fluency.

## Purpose

This repository keeps kata-style algorithm exercises. The goal is repeated practice, speed, and clean reasoning under interview constraints.

## Contents

- Java implementations of algorithm problems.
- Exercises that can be grouped by topic over time.
- Practice material for coding interviews.

## How To Use

Open the project in IntelliJ IDEA and run individual classes or tests while practicing.

## Status

Active learning repo. The next upgrade is to organize problems by pattern and add notes for complexity, invariant, and edge cases.

## Validation

For each kata, prefer tests or documented sample inputs that cover the normal case, edge cases, and complexity-relevant boundaries.

## Engineering Notes

A high-quality kata repo should make the invariant, time complexity, and space complexity visible near the implementation.
